## parent / child

태그간의 부모 자식 관계를 가지는 경우가 있다.

`<parent>`
    `<child></child>`
`</parent>`

<br>

#### 리스트의 경우

* ul : unordered list  /  ol : ordered list

```html
<ul>
  <li>1. HTML</li>
  <li>2. CSS</li>
  <li>3. JavaScript</li>
</ul>
```

```html
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
```

<br>

#### 표의 경우

* `<table>` 태그의 경우 3개가 같이 간다.

`<table>`
   `<tr>`
       `<td>head</td>`
       `<td>98.1%</td>`
    `</tr>` 
   `<tr>`
       `<td>body</td>`
       `<td>97.9%</td>`
    `</tr>`
    `<tr>`
        `<td>html</td>`
        `<td>97.9%</td>`
    `</tr>`
`</table>`
